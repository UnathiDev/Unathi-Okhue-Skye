Inventory Management System

This Inventory System is a robust desktop application built with Python and its powerful PyQt5 framework for the graphical user interface. It leverages SQLite as a lightweight and efficient database solution for seamless data management.

The system provides comprehensive functionalities for inventory control, including adding new items, viewing and searching existing stock, updating item details and managing user accounts with distinct roles. As a UNISA student passionate about sustainable technology and backend development, this project showcases my ability to develop a full-featured application, integrate database solutions and create intuitive user interfaces, ensuring efficient and reliable inventory operations.
